Setup

Connect to Google Sheets

Step 1 — Create a Google Sheet

Go to sheets.new in any browser. You don't need to do anything else to the sheet itself.

Step 2 — Open Apps Script

In your new Sheet, tap the menu Extensions → Apps Script. Delete whatever code is there.

Step 3 — Paste the backend code

Copy the entire contents of the file apps_script.js (provided alongside this app) and paste it into the Apps Script editor. Then tap Save.

Step 4 — Deploy as Web App

Tap Deploy → New deployment. Select type Web app. Set Execute as to Me and Who has access to Anyone. Tap Deploy and authorize it. Copy the Web App URL.

Step 5 — Paste the URL below

Paste your Web App URL here and tap Save. Share the same URL with your girlfriend — she opens this same HTML file and does the same step.
